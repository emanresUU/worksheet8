1) What is an array?

An array is a series of elements that have been given index numbers;
the index numbers can then be used to call and manipulate the elements.

2) How do we change the value of an element in an array?

We change the valueo of an element in an array by assigning a new value
using the elements index number. e.g.

var anArray = [1, 2, 3];
anArray[0] = "green";
anArray;
//-> ["green", 2, 3]
 
3) Which methods do we use to add an element to an array.

We use the push method and by assigning a value to a missing index.

4) What can an array hold? Can an array hold multiple types at once.

An array can hold each and every JavaScript data type in any combination.
These ar: numbers, booleans, strings, undefined values, and objects 
(including other arrays).
